# GAME NESR3

A multi-game platform with crypto integration (NES3 & DAF1).

## Games Included
- Eagle Game
- Car Game
- Airplane Game
- Forest Game

## Features
- Wallet for NES3 & DAF1
- Daily login bonus (3 DAF1 every 24h)
- Withdrawals possible only after reaching 10 NES3 / 10 DAF1
- Multi-language ready
- WebGL ready
