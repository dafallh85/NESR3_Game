# لعبة نسر 3 🚗💨

لعبة سيارات خيالية مربوطة بعملات BSC: NES3 و DAF1.

## كيفية التشغيل
1. افتح `index.html` في متصفح يدعم MetaMask.
2. اضغط على "ربط المحفظة".
3. يمكنك إيداع أو شراء NES3 / DAF1 أو سحب العملات من اللعبة إلى محفظتك.

## عقود العملات

- DAF1: 0x28227c3bb846f716cff8c7b4328dfc28fd3d3819
