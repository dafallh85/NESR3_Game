# 🦅 NESR3 Game

لعبة ويب HTML/CSS/JS خيالية – اختر عالمك (الجو، المدينة، البحار، الغابات، المدن)، اجمع النقاط، وحوّلها إلى عملات رقمية: NES3, DAF1, USDT, USDG.

## كيفية اللعب
1. اختر عالمك باستخدام الأزرار.  
2. اضغط "اجمع نقاط!" لكسب نقاط عشوائية.  
3. حول نقاطك إلى عملات رقمية باستخدام الأزرار المناسبة.  
4. يمكنك شحن العملات الرقمية مباشرة بالضغط على زر العملة.  
5. ترقية المستويات تحصل كل 50 نقطة.

## ملفات المشروع
- `index.html` → صفحة اللعبة الرئيسية  
- `style.css` → تصميم اللعبة  
- `script.js` → أكواد اللعبة والمنطق  
- `assets/` → صور العوالم (ستضاف لاحقاً)
